## Building from Source

Requirements:

-   [Deno](https://deno.land/) runtime

```bash
# Clone the repository
git clone https://github.com/yourusername/monokai-ghostty
cd monokai-ghostty

# Run the generation script
deno run --allow-read --allow-write generate.ts
```
